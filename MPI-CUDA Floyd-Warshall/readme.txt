1. Για compilation πατήστε make
2. Για εκτέλεση του προγράμματος για αριθμό διεργασιών μεγαλύερο του 1 πατήστε "mpirun -np num_of_processes ./program [n][p][w][kernel_type]"
   όπου kernel_type η υλοποιήση του πυρήνα που θα εκτελεστεί.
   0 -> Πυρήνας χωρίς τη χρήση κοινής μνήμης
   1 -> Πυρήνας με τη χρήση κοινής μνήμης
   2 -> Πυρήνας με τη χρήση πολλαπλών κελιών ανά thread.
3. Για την εκτέλεση του προγράμματος με μία διεργασία (τρίτη εργασία) πατήστε "./project3 [n][p][w]"
4. Για καθάρισμα πατήστε "make clean"
5. Αν επιθυμείτε να μεταβάλλετε τον αριθμό των κελιών ανά thread μεταβείτε στo αρχείο executeKernelMulElemPerThread.cu
   και αλλάξτε το "NUM_OF_THREADS_PER_BLOCK"
